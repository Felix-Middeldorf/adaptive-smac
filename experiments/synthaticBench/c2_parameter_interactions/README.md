# C2 Parameter Interactions

Initial experiments for SynthACticBench C2 using the `ackley` interaction
variant.

- `depth_policies/01_initial`: fixed depths 3, 6, 9, 12, 15, and 20.
- `leaf_policies/01_initial`: fixed leaf sizes 1, 2, 3, 4, and 5.

Both use five SMAC seeds, 1,000 trials, 10 instances, benchmark seed 52, and
10 random initial configurations.
